# Inspect Task Logs

Use task logs when you need to understand what happened during a Verft task run.

This page covers logs inside a task. For Docker container logs on the Verft server, see [Inspect Verft server logs](/inspect-logs).

## Before You Begin

You need access to the task.

## Steps

1. Open **Tasks**.

2. Select the task you want to inspect.

3. Review the task messages and status first.

4. Open the task logs or run output shown on the task detail page.

5. Check the most recent entries for failures, queued work, provider output, or terminal activity.

## Result

You can see task-level execution output from the browser and use it to understand task progress or failures.
