# Create Your First Task

Create a task to ask Verft to work on a repository or answer a repository question.

This guide is for users who have already added a repository and configured the required provider credentials.

## Before You Begin

You need:

- access to Verft
- permission to create tasks
- permission for the task type you want to create: **Build** or **Ask**
- at least one repository in **Repositories**
- provider credentials configured in **Settings**

## Steps

1. Open **Tasks**.

2. Select **New Task**.

3. Choose a **Repository**.

4. Select the **Task Type**:

   - **Build** for a task that can modify files.
   - **Ask** for a task that answers a repository question without writing files.

5. Choose the **Provider**, **Model**, and **Effort**.

6. For **Build** tasks, choose a **Branch Strategy**:

   - **Create feature branch** to work on a new branch.
   - **Work on existing branch** to work directly on the selected base branch.

7. Enter a short **Title**.

8. Enter the **Prompt** or **Question**.

9. Select **Create Task**.

## Result

Verft opens the task detail page. From there, you can follow messages, logs, task status, and any proposed changes.
